# Trading Tools — Roberto

Herramientas personales de análisis y mejora para trading de índices (Nasdaq / S&P500).

## Páginas

| Herramienta | Descripción |
|---|---|
| [Trading Dashboard](index.html) | Sube tu CSV de Exness y obtén diagnóstico de patrones problemáticos |
| [SMB Improvement Tracker](tracker.html) | Registro diario de sesiones y seguimiento de las 5 reglas de mejora |

## Cómo usar

1. Activa **GitHub Pages** en Settings → Pages → `main` → `/ (root)`
2. Tu URL será: `https://TU_USUARIO.github.io/TU_REPO/`
3. Accede desde el móvil y añade a pantalla de inicio para uso como PWA

## Problemas diagnosticados

- Stop-Loss inconsistente
- Aumento de tamaño de posición tras pérdidas
- Continuar operando tras rachas perdedoras (3+ stops)
